# Action – Unreal Engine Game Developer Portfolio

## 🚀 GitHub Pages Hosting

1. Create a repository named `action.github.io`
2. Upload all files
3. Go to Settings → Pages
4. Select branch: main
5. Save

Live URL:
https://action.github.io

## 🧪 Placeholders

- Images: Random images from picsum.photos
- Videos: Online video placeholders
- Hover over project images to preview gameplay

## 🔧 Customization

When ready, replace image/video URLs with your own.
